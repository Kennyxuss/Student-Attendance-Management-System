# Student-Attendance-Management-System

## Project Description
The **Student Attendance Management System** is a web-based application designed to modernize and simplify the process of recording, monitoring, and managing student attendance in educational institutions. It replaces traditional paper-based attendance sheets with a centralized digital platform that allows administrators to manage user accounts, instructors to create classes and record daily attendance, and authorized personnel to access accurate attendance records efficiently. The system securely stores attendance data, making it easy to retrieve attendance histories, monitor student participation, identify frequent absences or tardiness, and generate attendance reports for academic and administrative purposes.

## CRUD Operations
- **Create – Add students, classes, attendance records, and instructors.**
- **Read – View attendance history, student information, and class records.**
- **Update – Edit student details, class information, and attendance records.**
- **Delete – Remove students, classes, or incorrect attendance records.**
  
## Related Record Types
- **Students** – Student ID, name, course, year level.
- **Instructors** – Instructor ID, name, department.
- **Classes** – Class ID, subject, schedule, instructor.
- **Attendance** Records – Attendance ID, student, class, date, status (Present, Late, Absent).
